Semantic Tag

A semantic element clearly describes its meaning to both the browser and the developer. HTML5 tags are called semantic tag. They are-

  1. strong
  2. header
  3. footer
  4. article
  5. aside
  6. nav
  7. section

Inline Element

Inline elements display in a line. They do not force the text after them to a new line. They are-

  1. span
  2. bold(b)
  3. italic(i)
  4. anchor(a)

Block element

Block level elements take up as much space as possible by default. Each block level element will start a new line on the page, stacking down the page. In addition to stacking vertically, block level elements will also take up as much horizontal space as possible. They are-

  1. div
  2. paragraph tag(p)
  3. h1.....h6
  4. ul
  5. ol
